== Description ==
Plugin provides upcoming , past and all events list using shortcodes. 

* To list all events
`
[list_all_events]
` 

* To list upcoming events
`
[list_upcoming_events]
` 

* To list past events
`
[list_past_events]
` 

<strong>Features</strong>

* Shortcode based.
* Fully responsive and mobile ready.
* Easy input field for manage events.
* Easy to use.

== Installation ==
1. Upload the folder "simple-responsive-event-list" to "/wp-content/plugins/"
2. Activate the plugin through the "Plugins" menu in WordPress
3. Go to Wp-admin-> Events ->Add New , here we can add tab items.
4. use shortcode to list events in page or post

* To list all events
`
[list_all_events]
` 

* To list upcoming events
`
[list_upcoming_events]
` 

* To list past events
`
[list_past_events]
`
